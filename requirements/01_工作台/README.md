# 工作台模块（首页）

## 1. 模块定位

**首页/仪表盘**，配送员每天打开App的第一屏，聚合展示今日收入、待配送任务、快速开单入口。

**核心价值**：
- 3秒内知道今天赚了多少钱
- 5秒内知道今天还有多少家要送
- 10秒内完成快速开单

### 页面跳转关系

| 操作 | 跳转目标 | 说明 |
|-----|---------|------|
| 点击收入卡 | 展开趋势图（本页） | 显示本周收入趋势 |
| 点击"开始配送" | 地图页面 | 进入配送导航模式 |
| 点击"拨打电话" | 系统拨号 | 唤起手机拨号界面 |
| 点击"更多" | 底部弹出菜单 | 修改/取消/备注 |
| 点击"查看全部待送" | 订单页面 | 筛选待配送订单 |
| 点击"快速开单" | 底部弹出选择客户 | 不跳转，本页完成 |
| 点击底部导航"客户" | 客户列表页面 | 查看所有客户 |
| 点击底部导航"我的" | 个人中心页面 | 收入详情、设置 |

## 5. 数据来源

| 展示内容 | 来源模块 | 数据接口 | 更新频率 |
|---------|---------|---------|---------|
| 今日已收 | 财务模块 | GET /api/finance/today-income | 进入页面刷新 |
| 今日待收 | 财务模块 | GET /api/finance/today-receivable | 进入页面刷新 |
| 本周趋势 | 财务模块 | GET /api/finance/weekly-trend | 展开时刷新 |
| 待配送订单 | 订单模块 | GET /api/orders?status=pending | 进入页面刷新 |
| 客户信息 | 客户模块 | GET /api/customers/:id | 已缓存 |

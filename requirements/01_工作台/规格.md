
**状态定义**：

| 状态 | 高度 | 显示内容 | 可交互元素 |
|-----|------|---------|-----------|
| 折叠态 | 120dp | 今日已收（36sp）、今日待收（14sp） | 收入卡（点击展开） |
| 展开动画中 | 120dp→320dp | 同上 + 趋势图淡入 | 不可交互 |
| 展开态 | 320dp | 折叠态内容 + 趋势图 + 每日明细 | 趋势图点选、收入卡（点击折叠） |

**状态转换动画参数**：
- 高度过渡：300ms，ease-in-out
- 内容淡入：200ms，delay 100ms
- 趋势图绘制：500ms，从左到右

## 3. 边界条件

### 3.1 收入计算边界

| 场景 | 输入 | 预期输出 | 处理方式 |
|-----|------|---------|---------|
| 今日无订单 | 订单数=0 | 今日已收=¥0 | 显示"今天还没有订单"，不显示0 |
| 金额有小数 | 应收=¥135.5 | 显示¥135.5 | 保留1位小数，使用Decimal计算 |
| 金额为负数 | 退款场景 | 显示-¥50 | 红色标记，今日已收减少 |
| 金额超过1万 | 已收=¥12,500 | 显示¥1.25万 | >10000时单位转换，保留2位小数 |
| 计算精度溢出 | 浮点数精度问题 | 四舍五入到分 | 后端使用Decimal(10,2)类型 |
| 跨天交易 | 23:59开单，00:05完成 | 归属到完成日期 | 以交易完成时间为准统计 |

### 3.2 客户卡片边界

| 场景 | 输入 | 预期输出 | 处理方式 |
|-----|------|---------|---------|
| 客户姓名超长 | "李..."（20字） | 截断显示"李大..." | 最多显示6个汉字，超出用... |
| 地址超长 | 50字地址 | 单行截断15字 | 单行显示，超出省略 |
| 金额超长 | ¥123,456.78 | 正常显示 | 千分位分隔符 |
| 无待配送订单 | 待送数=0 | 显示"今日任务已完成" | 显示庆祝图标，鼓励文案 |
| 待送超过99家 | 待送数=100+ | 显示"99+" | 右上角红色角标 |
| 欠瓶超过99个 | 欠瓶=150个 | 显示"99+" | 红色角标，点击查看明细 |
| 客户无地址 | 地址字段为空 | 显示"地址未填写" | 灰色提示，提醒完善 |
| 客户无电话 | 电话字段为空 | 隐藏电话按钮 | 显示"需补充电话" |

### 3.3 快速开单边界

| 场景 | 输入 | 预期输出 | 处理方式 |
|-----|------|---------|---------|
| 无客户可选 | 新客户首次使用 | 显示"+添加新客户" | 引导创建客户 |
| 搜索无结果 | 搜索"张三"不存在 | 显示"未找到，点击添加" | 提供快捷添加入口 |
| 库存不足 | 15kg重瓶=0，开15kg单 | 提示"15kg重瓶库存不足" | 建议改规格或"稍后配送" |
| 单价为0 | 输入¥0 | 允许，但二次确认 | "确认免费赠送？" |
| 数量超过99 | 输入100 | 限制最大99 | Toast"单次最多99瓶" |
| 连续快速点击 | 1秒内点击5次确认 | 只响应第1次 | 按钮防抖300ms，loading状态 |
| 网络中断时开单 | 点击确认时断网 | 本地记录，提示离线 | 乐观显示，后台同步 |

### 3.4 当场完成边界

| 场景 | 输入 | 预期输出 | 处理方式 |
|-----|------|---------|---------|
| 实收为0 | 输入¥0 | 标记为"记账" | 自动切换收款方式为"记账" |
| 实收大于应收 | 应收¥130，实收¥200 | 提示"多收¥70，是否转为存款？" | 提供转存款选项 |
| 实收小于应收 | 应收¥130，实收¥100 | 标记为"部分收款"，欠款¥30 | 自动生成欠款记录 |
| 收款方式未选 | 点击完成未选方式 | 提示"请选择收款方式" | 收款方式必填 |
| 5秒内撤销 | 点击撤销按钮 | 回退到编辑状态 | 库存回滚，收入扣除 |
| 超过5秒撤销 | 撤销按钮消失 | 不可撤销 | 只能通过"修改订单"调整 |

## 4. 手势交互

### 4.1 首页全局手势

| 手势 | 作用区域 | 效果 | 参数 |
|-----|---------|------|------|
| 下拉 | 页面顶部（距离顶部<100px） | 刷新数据 | 触发距离：80px，loading高度：60px |
| 下拉 | 页面其他区域 | 无效果 | 防止误触 |
| 上滑 | 客户列表区域 | 加载更多待送 | 距离底部20px触发，loading在底部 |
| 左滑 | 客户卡片 | 显示"拨打电话"按钮 | 触发：>100px，最大：150px |
| 右滑 | 客户卡片 | 显示"标记完成"按钮 | 同上 |
| 长按 | 客户卡片 | 显示更多操作菜单 | 触发时间：500ms |
| 双击 | 收入卡 | 快速展开/折叠趋势图 | 两次点击间隔<300ms |
| 捏合 | 趋势图区域 | 缩放趋势图时间范围 | 双指捏合，范围：7天→30天 |

### 4.2 底部面板手势

| 手势 | 效果 | 参数 |
|-----|------|------|
| 下滑 | 关闭面板 | 触发：>100px，速度>200px/s，动画：向下滑出200ms |
| 上滑 | 面板全屏展开 | 触发：>50px，高度：屏幕80%，动画：300ms |
| 点击面板外遮罩 | 关闭面板 | 遮罩透明度：50%黑色，点击响应区域：除面板外全部 |
| 点击面板内关闭按钮 | 关闭面板 | 按钮位置：右上角，尺寸：24dp |
| 面板内左右滑动 | 切换客户（选择客户面板） | 卡片宽度：屏幕宽-32dp，切换动画：200ms |

### 4.3 数字输入手势

| 手势 | 作用区域 | 效果 | 参数 |
|-----|---------|------|------|
| 长按+/- | 数量步进器 | 快速增减 | 长按100ms后开始连续增减，频率：10次/秒 |
| 双击输入框 | 金额输入框 | 全选数字 | 方便快速修改 |
| 键盘下滑 | 键盘区域 | 收起键盘 | 响应区域：键盘上方20px |

## 5. 动画效果

### 5.1 页面加载动画

| 元素 | 动画 | 时长 | 缓动函数 | 延迟 |
|-----|------|------|---------|------|
| 骨架屏 | 闪烁动画（shimmer） | 1500ms | linear | 0ms，循环 |
| 收入数字 | 从0滚动到实际值 | 800ms | ease-out | 200ms |
| 收入数字小数 | 小数位单独滚动 | 400ms | ease-out | 600ms |
| 客户卡片1 | 从下往上淡入，位移20px | 300ms | ease-out | 400ms |
| 客户卡片2 | 同上 | 300ms | ease-out | 500ms |
| 客户卡片3 | 同上 | 300ms | ease-out | 600ms |
| 快速开单按钮 | 从底部弹入，位移40px | 400ms | spring | 700ms |
| 待送数量角标 | 缩放弹跳（0→1.2→1） | 400ms | spring | 800ms |

**收入数字滚动实现**：
```
起始值：0
结束值：1680
时长：800ms
插值器：DecelerateInterpolator(1.5f)
显示格式：¥{当前值，整数部分}
小数部分：单独动画，600ms开始，400ms完成
```

### 5.2 交互动画

| 交互 | 动画 | 时长 | 缓动函数 | 说明 |
|-----|------|------|---------|------|
| 点击收入卡展开 | 高度：120dp→320dp | 300ms | ease-in-out | 内容淡入delay 100ms |
| 点击收入卡折叠 | 高度：320dp→120dp | 250ms | ease-in-out | 内容先淡出 |
| 点击快速开单按钮 | 缩放：1→0.95→1 | 150ms | ease-out | 触觉反馈（轻微震动） |
| 面板弹出 | 从底部：translateY(100%)→0 | 250ms | ease-out | 遮罩：透明度0→0.5，200ms |
| 面板关闭 | 向底部：translateY(0)→100% | 200ms | ease-in | 遮罩先淡出 |
| 面板全屏 | 高度：50%→80% | 300ms | ease-in-out | 内容重新布局 |
| 客户卡片左滑 | translateX：0→-100px | 150ms | ease-out | 超过100px弹性回弹 |
| 客户卡片回弹 | translateX：-100px→0 | 200ms | spring | 弹性系数：0.5 |
| Toast提示 | 淡入+上移10px | 200ms | ease-out | 停留2s，淡出200ms |
| 收入更新跳动 | 缩放：1→1.1→1 | 300ms | spring | 颜色闪烁：绿→正常 |

### 5.3 状态变化动画

| 状态变化 | 动画 | 时长 | 说明 |
|---------|------|------|------|
| 库存不足提示 | 按钮抖动（X轴±5px，3次） | 300ms | 红色闪烁提示 |
| 订单创建成功 | 对勾绘制（SVG path动画） | 500ms | 随后Toast淡入 |
| 撤销成功 | 内容回滚+数字回滚 | 400ms | 逆向动画 |
| 离线模式提示 | 顶部bar滑入 | 200ms | 黄色背景，显示待同步数 |
| 同步完成 | 提示更新为"已同步"，2s后滑出 | 200ms+2000ms+200ms | 绿色对勾 |

### 5.4 连续动画参数

**收入数字滚动详细参数**：
```kotlin
// Android示例
val animator = ValueAnimator.ofInt(0, targetAmount)
animator.duration = 800
animator.interpolator = DecelerateInterpolator(1.5f)
animator.addUpdateListener {
    val value = it.animatedValue as Int
    textView.text = "¥${NumberFormat.getInstance().format(value)}"
}
animator.startDelay = 200
animator.start()
```

**Spring动画参数（面板关闭回弹）**：
```kotlin
SpringAnimation(view, DynamicAnimation.TRANSLATION_Y, 0f).apply {
    spring.dampingRatio = SpringForce.DAMPING_RATIO_MEDIUM_BOUNCY
    spring.stiffness = SpringForce.STIFFNESS_LOW
    start()
}
```

## 7. 性能规格

### 7.1 加载性能

| 指标 | 目标值 | 测试条件 | 优化方案 |
|-----|--------|---------|---------|
| 首屏加载时间 | < 1s | 4G网络，缓存数据 | 骨架屏+本地缓存优先 |
| 首屏加载时间 | < 2s | 4G网络，首次加载 | 懒加载非关键资源 |
| 数据刷新时间 | < 1.5s | 下拉刷新 | 增量更新，只刷新变化部分 |
| 面板弹出时间 | < 200ms | 本地渲染 | 预渲染面板，动画优化 |
| 收入数字滚动 | 60fps | 动画过程中 | 使用硬件加速 |

### 7.2 运行时性能

| 指标 | 目标值 | 监控方案 |
|-----|--------|---------|
| 页面滚动帧率 | > 55fps | 全程监控，低于50fps上报 |
| 内存占用 | < 80MB | 定时检测，超阈值清理缓存 |
| CPU占用 | < 25% | 后台任务合并，减少唤醒 |
| 电量消耗 | < 5%/小时 | 后台同步间隔优化 |

### 7.3 缓存策略

| 数据类型 | 缓存时间 | 更新策略 | 存储方式 |
|---------|---------|---------|---------|
| 今日收入 | 3分钟 | 进入页面刷新，后台3分钟刷新 | LocalStorage |
| 待配送列表 | 2分钟 | 进入页面刷新，下拉刷新 | LocalStorage |
| 客户列表 | 10分钟 | 搜索时实时，列表用缓存 | IndexedDB |
| 趋势数据 | 1小时 | 展开趋势图时刷新 | LocalStorage |
| 离线交易队列 | 永久 | 网络恢复立即同步 | IndexedDB |

### 7.4 离线性能

| 场景 | 性能要求 | 处理方案 |
|-----|---------|---------|

| 离线查询 | < 200ms | 本地数据库查询 |
| 批量同步 | < 3s（10笔） | 后台批量提交，冲突处理 |

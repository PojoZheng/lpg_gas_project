# 财务模块需求文档

## 1. 模块定位

**一句话描述**：记录收入支出，计算利润，日结对账。

**核心价值**：
- 知道今天赚了多少
- 知道这个月赚了多少
- 欠款清晰，及时催收

---

## 2. 入口（与首页对齐，无底部tab）

**产品经理说明**：财务是「查看+对账」功能，不是高频操作。配送员每天配送结束后看一眼今天赚了多少，首页卡片已经满足80%的需求。

| 入口 | 来源 | 说明 | 使用场景 |
|-----|------|------|---------|
| **首页收入卡片** | 首页-今日已收/待收 | 点击展开财务详情 | 每天看收入（最常用） |
| **我的-财务管理** | 我的页面 | 日结对账、利润统计、历史查询 | 深度财务管理 |
| **我的-债务管理** | 我的页面 | 欠瓶/欠款管理、催收 | 管理债务时 |
| **订单完成时** | 自动记录 | 自动记账 | 无需操作 |

## 3. 收入记录

### 3.1 收入来源

| 收入类型 | 说明 | 来源 |
|---------|------|------|
| **气款** | 液化气费用 | 订单完成 |
| **押金** | 新客收押金 | 订单完成 |
| **租金** | 气瓶租金 | 订单完成/按月扣 |
| **残液** | 残液价值 | 收空瓶时 |

```
│                                         │
```

```
│  ─────────────────────                  │
│  净利润        +¥330                    │

│  完成订单      8单                      │
│  配送气瓶      15kg×6, 10kg×2           │
│  收回空瓶      15kg×5, 10kg×1           │
│  产生欠瓶      2个                      │
│                                         │
├─────────────────────────────────────────┤
│  【收款方式】                            │
│  现金          ¥980                     │
│  微信          ¥450                     │

│  记账（欠款）  ¥100                     │
│                                         │
├─────────────────────────────────────────┤
│  【对账确认】                            │
│  [现金盘点]  实收¥980  = 系统¥980  ✓    │
│  [微信对账]  实收¥450  = 系统¥450  ✓    │
│                                         │
│     [  确 认 日 结  ]                   │
│                                         │
└─────────────────────────────────────────┘
```

### 5.2 对账流程

```
收工后
    ↓
查看今日日结
    ↓
核对现金：数一下手里现金和系统是否一致
    ↓
核对微信/支付宝：看收款记录
    ↓
确认无误 → 点击[确认日结]
    ↓
日结完成，生成日结记录
```

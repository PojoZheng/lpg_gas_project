## 2. IncomeRecord 收入记录

### 2.1 表结构

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | STRING(32) | 是 | UUID | 主键 |
| dealer_id | STRING(32) | 是 | - | 配送员ID |
| customer_id | STRING(32) | 否 | NULL | 客户ID |
| order_id | STRING(32) | 否 | NULL | 关联订单ID |
| income_type | ENUM | 是 | - | 收入类型 |
| spec | STRING(10) | 否 | NULL | 规格 |
| quantity | INTEGER | 否 | NULL | 数量 |
| unit_price | DECIMAL(10,2) | 否 | NULL | 单价 |
| amount | DECIMAL(10,2) | 是 | - | 金额 |
| pay_method | ENUM | 是 | - | 支付方式 |
| pay_time | DATETIME | 否 | NULL | 支付时间 |
| transaction_no | STRING(100) | 否 | NULL | 交易流水号 |
| remark | STRING(200) | 否 | NULL | 备注 |
| income_date | DATE | 是 | CURRENT | 收入日期 |
| created_at | DATETIME | 是 | CURRENT | 创建时间 |

### 2.2 枚举定义

**收入类型 (income_type)**：
```
GAS             = '气款'        // 燃气销售款
DEPOSIT         = '押金'        // 气瓶押金
RENT            = '租金'        // 气瓶租金
RESIDUAL        = '残液'        // 残液收入
DELIVERY_FEE    = '配送费'      // 配送费用
OWED_CONVERT    = '欠瓶转收入'   // 欠瓶转为收入
OTHER           = '其他'
```

**支付方式 (pay_method)**：
```
WECHAT          = '微信支付'
ALIPAY          = '支付宝'
CASH            = '现金'
CREDIT          = '赊账'        // 欠款
OWED_OFFSET     = '欠瓶抵扣'     // 用欠瓶抵扣
```

### 4.2 枚举定义

```

CONFIRMED       = '已对账'
ADJUSTED        = '已调整'
```

### 4.3 计算公式

```javascript
// 收入合计
income_total = income_gas + income_deposit + income_rent + 
               income_residual + income_delivery

// 支出合计
expense_total = expense_purchase + expense_deposit_return + 
                expense_rent_return + expense_delivery + expense_loss

// 毛利（仅气款）
// 需要从订单明细计算
// 毛利 = 气款收入 - Σ(各规格销量 × 对应进价)

// 净利
net_profit = income_total - expense_total

// 利润率
profit_margin = (net_profit / income_total) × 100
```

## 7. 视图/查询

### 7.1 今日财务汇总查询

```sql
-- 今日收入汇总
SELECT 
  income_type,
  SUM(amount) as total_amount,
  COUNT(*) as count
FROM income_record
WHERE dealer_id = ? 
  AND income_date = CURRENT_DATE
GROUP BY income_type

-- 今日支出汇总
SELECT 
  expense_type,
  SUM(amount) as total_amount,
  COUNT(*) as count
FROM expense_record
WHERE dealer_id = ? 
  AND expense_date = CURRENT_DATE
GROUP BY expense_type
```

### 7.2 利润分析查询

```sql
-- 按月统计利润
SELECT 
  DATE_FORMAT(income_date, '%Y-%m') as month,
  SUM(income_total) as total_income,
  SUM(expense_total) as total_expense,
  SUM(net_profit) as total_profit,
  AVG(profit_margin) as avg_margin
FROM daily_settlement
WHERE dealer_id = ?
  AND settlement_date >= DATE_SUB(CURRENT_DATE, INTERVAL 12 MONTH)
GROUP BY month
ORDER BY month DESC
```

### 7.3 债务预警查询

```sql
-- 超期债务查询
SELECT 
  d.*,
  c.name as customer_name,
  c.phone as customer_phone,
  DATEDIFF(CURRENT_DATE, d.owed_date) as overdue_days
FROM debt_record d
JOIN customer c ON d.customer_id = c.id
WHERE d.dealer_id = ?
  AND d.status = 'OWING'
  AND DATEDIFF(CURRENT_DATE, d.owed_date) > d.remind_days
ORDER BY overdue_days DESC
```

## 8. 索引设计

| 表名 | 索引名 | 字段 | 类型 |
|-----|-------|------|------|
| income_record | idx_dealer_date_type | dealer_id, income_date, income_type | 普通索引 |
| income_record | idx_order | order_id | 普通索引 |
| income_record | idx_customer | customer_id | 普通索引 |
| expense_record | idx_dealer_date_type | dealer_id, expense_date, expense_type | 普通索引 |
| expense_record | idx_purchase | purchase_id | 普通索引 |
| daily_settlement | idx_dealer_date | dealer_id, settlement_date | 唯一索引 |
| debt_record | idx_dealer_customer | dealer_id, customer_id, status | 普通索引 |
| debt_record | idx_status_date | dealer_id, status, owed_date | 普通索引 |
| debt_repay_record | idx_debt | debt_id | 普通索引 |

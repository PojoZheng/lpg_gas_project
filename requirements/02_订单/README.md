# 订单模块

## 1. 模块定位

订单模块是系统的核心业务模块，负责管理从开单到配送完成的全流程。

---

## 3. 核心功能列表

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 创建订单 | 含押金/租金/残液/欠瓶处理 | P0 |
| 批量开单 | 复制历史订单快速生成 | P0 |
| 订单管理 | 列表、详情、状态流转 | P0 |
| 完成收款 | 送达确认、收款、修改金额 | P0 |
| 欠瓶管理 | 记录、提醒、追收 | P0 |
| 残液记录 | 收空瓶时记录残液 | P0 |

---

## 4. 与其他模块的关系

### 被依赖的模块

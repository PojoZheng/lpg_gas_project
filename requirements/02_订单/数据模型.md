# 订单模块数据模型

## 1. Order（订单主表）

| 字段 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| id | string | 是 | 订单ID（UUID） |
| dealer_id | string | 是 | 气贩子ID |
| customer_id | string | 是 | 客户ID |
| address | string | 是 | 配送地址 |
| order_date | datetime | 是 | 下单时间 |
| delivery_date | datetime | 否 | 配送时间 |
| status | enum | 是 | 待配送/已完成/已取消 |
| total_gas_amount | decimal | 是 | 气款合计 |
| gas_fee | decimal | 是 | 气费 |
| deposit_amount | decimal | 是 | 押金 |
| rent_amount | decimal | 是 | 租金 |
| residual_deduction | decimal | 是 | 残液抵扣 |
| total_amount | decimal | 是 | 应收合计 |
| paid_amount | decimal | 是 | 实收金额 |
| payment_method | enum | 否 | 现金/微信/支付宝/记账 |
| owed_cylinder_action | enum | 否 | 欠瓶处理方式 |
| remark | string | 否 | 备注 |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |

### 状态枚举

```
status:
  - pending      # 待配送
  - completed    # 已完成
  - cancelled    # 已取消
```

---

## 2. OrderItem（订单项表）

| 字段 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| id | string | 是 | 订单项ID |
| order_id | string | 是 | 关联订单ID |
| cylinder_spec | string | 是 | 气瓶规格（15kg/10kg/50kg） |
| quantity | int | 是 | 数量 |
| unit_price | decimal | 是 | 单价 |
| amount | decimal | 是 | 金额（quantity × unit_price） |

---

## 3. EmptyCylinderRecord（收空瓶记录）

| 字段 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| id | string | 是 | 记录ID |
| order_id | string | 是 | 关联订单ID |
| cylinder_spec | string | 是 | 气瓶规格 |
| quantity | int | 是 | 收回数量 |
| residual_kg | decimal | 否 | 残液重量（kg） |
| residual_value | decimal | 否 | 残液价值 |

---

## 4. OwedCylinder（欠瓶记录）

| 字段 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| id | string | 是 | 记录ID |
| dealer_id | string | 是 | 气贩子ID |
| customer_id | string | 是 | 客户ID |
| order_id | string | 是 | 产生欠瓶的订单ID |
| cylinder_spec | string | 是 | 气瓶规格 |
| quantity | int | 是 | 欠瓶数量 |
| status | enum | 是 | 未收回/已收回/转押金/作废 |
| created_at | datetime | 是 | 欠瓶产生时间 |
| recovered_at | datetime | 否 | 收回时间 |
| converted_type | enum | 否 | 转化类型：deposit/owed |
| related_owed_record_id | string | 否 | 关联欠款记录ID |
| days_owed | int | 是 | 欠瓶天数（自动计算） |

### 状态枚举

```
status:
  - outstanding   # 未收回
  - recovered     # 已收回
  - converted     # 转为收入
  - waived        # 作废
```

## 5. SQL建表语句

```sql
-- 订单主表
CREATE TABLE orders (
    id VARCHAR(32) PRIMARY KEY,
    dealer_id VARCHAR(32) NOT NULL,
    customer_id VARCHAR(32) NOT NULL,
    address VARCHAR(255) NOT NULL,
    order_date DATETIME NOT NULL,
    delivery_date DATETIME,
    status VARCHAR(20) NOT NULL DEFAULT 'pending',
    total_gas_amount DECIMAL(10,2) NOT NULL DEFAULT 0,
    gas_fee DECIMAL(10,2) NOT NULL DEFAULT 0,
    deposit_amount DECIMAL(10,2) NOT NULL DEFAULT 0,
    rent_amount DECIMAL(10,2) NOT NULL DEFAULT 0,
    residual_deduction DECIMAL(10,2) NOT NULL DEFAULT 0,
    total_amount DECIMAL(10,2) NOT NULL DEFAULT 0,
    paid_amount DECIMAL(10,2) NOT NULL DEFAULT 0,
    payment_method VARCHAR(20),
    owed_cylinder_action VARCHAR(20),
    remark TEXT,
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_dealer (dealer_id),
    INDEX idx_customer (customer_id),
    INDEX idx_status (status),
    INDEX idx_order_date (order_date)
);

-- 订单项表
CREATE TABLE order_items (
    id VARCHAR(32) PRIMARY KEY,
    order_id VARCHAR(32) NOT NULL,
    cylinder_spec VARCHAR(10) NOT NULL,
    quantity INT NOT NULL DEFAULT 1,
    unit_price DECIMAL(10,2) NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    FOREIGN KEY (order_id) REFERENCES orders(id),
    INDEX idx_order (order_id)
);

-- 欠瓶记录表
CREATE TABLE owed_cylinders (
    id VARCHAR(32) PRIMARY KEY,
    dealer_id VARCHAR(32) NOT NULL,
    customer_id VARCHAR(32) NOT NULL,
    order_id VARCHAR(32) NOT NULL,
    cylinder_spec VARCHAR(10) NOT NULL,
    quantity INT NOT NULL DEFAULT 1,
    status VARCHAR(20) NOT NULL DEFAULT 'outstanding',
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    recovered_at DATETIME,
    converted_type VARCHAR(20),
    related_owed_record_id VARCHAR(32),
    INDEX idx_dealer (dealer_id),
    INDEX idx_customer (customer_id),
    INDEX idx_status (status),
    INDEX idx_created (created_at)
);
```

# 订单模块接口

## 1. 订单管理

### 1.1 创建订单

```http
POST /api/orders
Content-Type: application/json

{
  "customer_id": "C001",
  "address": "阳光花园5栋",
  "cylinders": [
    {"spec": "15kg", "quantity": 1, "price": 120}
  ],
  "gas_fee": 10,
  "deposit": 0,
  "rent": 0,
  "residual_deduction": 5,
  "empty_cylinders": [
    {"spec": "15kg", "quantity": 1, "residual_kg": 1}
  ],
  "owed_cylinder_action": "collect"
}
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "id": "O202604070001",
    "status": "pending",
    "total_amount": 272.5,
    "created_at": "2026-04-07T10:30:00Z"
  }
}
```

---

### 1.2 获取订单列表

```http
GET /api/orders?status=pending&page=1&size=20
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "total": 156,
    "page": 1,
    "size": 20,
    "list": [
      {
        "id": "O202604070001",
        "customer_name": "李大妈",
        "address": "阳光花园5栋",
        "status": "pending",
        "total_amount": 272.5,
        "order_date": "2026-04-07T10:30:00Z"
      }
    ]
  }
}
```

---

### 1.3 获取订单详情

```http
GET /api/orders/{id}
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "id": "O202604070001",
    "customer": {
      "id": "C001",
      "name": "李大妈",
      "phone": "138xxxx1234"
    },
    "address": "阳光花园5栋",
    "status": "completed",
    "items": [...],
    "amounts": {...},
    "created_at": "2026-04-07T10:30:00Z"
  }
}
```

---

### 1.4 完成订单（送达并收款）

```http
POST /api/orders/{id}/start-delivery
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "id": "O202604070001",
    "status": "delivering",
    "start_time": "2026-04-07T11:00:00Z"
  }
}
```

---

### 1.5 送达确认

```http
POST /api/orders/{id}/confirm-delivery
Content-Type: application/json

{
  "empty_cylinders": [
    {"spec": "15kg", "quantity": 1, "residual_kg": 0.5}
  ],
  "remark": ""
}
```

---

### 1.6 取消订单

```http
POST /api/orders/{id}/cancel
Content-Type: application/json

{
  "reason": "客户取消"
}
```

---

## 2. 欠瓶管理

### 2.1 获取欠瓶列表

```http
GET /api/owed-cylinders?dealer_id=D001
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "total": 23,
    "total_value": 3450,
    "by_days": {
      "within_7": 5,
      "day_8_30": 12,
      "over_30": 6
    },
    "list": [...]
  }
}
```

---

### 2.2 标记欠瓶已收回

```http
POST /api/owed-cylinders/{id}/recover
```

---

### 2.3 欠瓶转为收入

```http
POST /api/owed-cylinders/{id}/convert
Content-Type: application/json

{
  "type": "deposit",  // deposit:用押金抵扣, owed:转为欠款
  "remark": ""
}
```

---

## 3. 批量开单

### 3.1 获取历史订单

```http
GET /api/orders/history?date=2026-04-06
```

---

### 3.2 批量创建订单

```http
POST /api/orders/batch
Content-Type: application/json

{
  "source_date": "2026-04-06",
  "order_ids": ["O202604060001", "O202604060002"]
}
```

---

## 4. 错误码

| 错误码 | 说明 |
|-------|------|
| 1001 | 客户不存在 |
| 1002 | 库存不足 |
| 1003 | 订单状态不允许此操作 |
| 1004 | 欠瓶处理失败 |
| 1005 | 押金计算错误 |

# 客户模块规格文档

## 1. 页面清单

| 页面 | 路径 | 说明 |
|-----|------|------|
| 客户列表页 | `/customers` | 客户列表、搜索、筛选 |
| 客户详情页 | `/customers/:id` | 客户档案、订单历史、财务信息 |

| 押金账户页 | `/customers/:id/deposit` | 押金/租金账户详情 |
| 欠瓶记录页 | `/customers/:id/owed` | 欠瓶/欠款记录 |

### 2.4 交互说明

#### 搜索交互

```
点击搜索栏
    ↓
唤起键盘，光标闪烁
    ↓
输入文字（实时搜索，延迟300ms）
    ↓
显示匹配结果（高亮匹配文字）
    ↓
无结果时显示"无匹配客户，去新增？"
```

**搜索规则**：
- 匹配字段：姓名、电话、地址、备注
- 排序：最近有订单的优先
- 防抖：300ms

#### 筛选交互

| 筛选项 | 说明 |
|-------|------|
| 全部 | 显示所有客户 |
| 住宅 | 客户类型=住宅 |
| 商铺 | 客户类型=商铺 |
| 餐馆 | 客户类型=餐馆 |
| 有欠瓶 | 欠瓶数量>0 |
| 有欠款 | 欠款金额>0 |
| 最近下单 | 7天内有订单 |
| 长期未送 | 30天无订单 |

#### 客户卡片交互

| 操作 | 触发 | 反馈 |
|-----|------|------|

## 7. 状态定义

### 7.1 客户状态

```
客户状态机：

正常
  ↓
┌─────────────────────────────────────┐
│ 预警状态（可同时存在多个）：         │
│ • 有欠瓶（欠瓶数>0）                │
│ • 有欠款（欠款金额>0）               │
│ • 欠瓶超期（欠瓶天数>提醒天数）      │
│ • 欠款超期（欠款天数>提醒天数）      │
│ • 长期未送（30天无订单）             │
└─────────────────────────────────────┘
  ↓
停用（客户不再配送，可恢复）
```

### 7.2 欠瓶状态

| 状态 | 说明 | 触发条件 |
|-----|------|---------|
| 欠瓶中 | 客户欠空瓶 | 送达时客户无空瓶 |
| 已还 | 已归还空瓶 | 下次配送时归还 |
| 已转收入 | 转为收入 | 选择"转为收入" |
| 已免除 | 放弃追讨 | 选择"免除" |

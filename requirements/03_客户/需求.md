# 客户模块需求文档

## 1. 模块定位

**一句话描述**：管理客户档案、押金租金账户、历史订单、欠款欠瓶。

**核心价值**：
- 客户信息不丢失
- 押金租金清晰可查
- 欠款欠瓶自动提醒

---

## 2. 入口

| 入口 | 来源 | 说明 |
|-----|------|------|
| 底部导航-客户 | 首页底部 | 客户列表入口 |
| 首页-快速开单 | 选客户时 | 查看/选择客户 |
| 订单详情 | 点击客户名 | 查看客户详情 |
| 我的-客户管理 | 我的页面 | 管理全部客户 |

### 4.2 押金/租金账户

**账户类型**：
| 账户 | 说明 | 操作 |
|-----|------|------|
| 押金账户 | 按规格分别管理 | 收、退、扣 |
| 租金账户 | 从押金扣或单独收 | 收、扣 |
| 预存账户 | 客户预存气款 | 充值、扣款 |

**交易记录**：
- 每笔收退记录可查
- 关联订单显示

## 6. 新增/编辑客户

### 6.1 客户信息字段

| 字段 | 必填 | 说明 |
|-----|------|------|
| 姓名 | 是 | 客户称呼 |
| 电话 | 是 | 联系电话 |
| 地址 | 是 | 配送地址 |
| 客户类型 | 是 | 居民/商户/工业 |
| 标签 | 否 | VIP/免押金/免租金等 |
| 备注 | 否 | 其他信息 |
| 地图标记 | 否 | 地址坐标（用于导航） |

### 6.2 智能识别

**拍照识别**：
- 拍客户名片/门牌，自动识别姓名、电话、地址
- 减少手动输入

## 核心业务流程

### 1. 配送主流程（高频，每天20-50次）

```
首页「待送列表」→ 点击订单 → 导航配送 → 送达确认 → 安检（10秒） → 完成
                                    ↓
                              首页「快速开单」← 新建订单
```

### 2. 开单流程（高频）

```
首页「快速开单」→ 选择客户 → 确认订单 → 选择类型
                                    ↓
                    ┌───────────────┼───────────────┐
                    ▼               ▼               ▼
                稍后配送        当场完成         预约配送
                    ↓               ↓               ↓
                锁定库存      扣减库存+收款     加入预约列表
                    ↓               ↓               ↓
                加入待送列表   直接完成         等待提醒
```

### 3. 日结流程（每天1次）

```
配送结束
    ↓
首页收入卡片看一眼
    ↓
点击卡片进入财务
    ↓
核对今日收支
    ↓
确认无误标记已对账
```

## 关键业务规则（以首页为准）

### 导航规则

| 规则 | 说明 |
|-----|------|
| 底部Tab | 仅3个：首页、客户、我的 |
| 订单历史 | 不在底部Tab，从「我的-订单历史」进入 |
| 库存/安检/财务 | 无底部Tab，从首页快捷入口或「我的」进入 |

### 订单规则

| 规则 | 说明 |
|-----|------|
| 两状态制 | 只有「待配送」和「已完成」，没有「配送中」 |
| 订单修改 | 24小时内可改规格数量，5秒内可撤销 |
| 连续开单 | 订单完成后不关闭面板，方便批量开单 |
| 离线开单 | 无网时可开单，自动同步 |

### 库存规则

| 规则 | 说明 |
|-----|------|
| 锁定机制 | 「稍后配送」订单锁定库存 |
| 扣减时机 | 完成时扣减，当场完成立即扣减 |
| 预警展示 | 首页显示库存状态表情（😊😐😰） |
| 入库简化 | 默认只录数量价格，编号选填 |

### 安检规则

| 规则 | 说明 |
|-----|------|
| 触发时机 | 订单完成后自动弹出（主入口） |
| 快速完成 | 默认全部正常，只需拍照，10秒内完成 |
| AI识别 | 自动识别气瓶号，无需手输 |
| 上报机制 | 后台自动上报，配送员无感知 |

### 财务规则

| 规则 | 说明 |
|-----|------|
| 自动记账 | 订单数据自动生成财务记录 |
| 日结对账 | 每天确认一次，标记已对账 |
| 首页卡片 | 显示今日已收/待收，点击进入详情 |
| 债务管理 | 从我的页面进入，统一管欠瓶/欠款 |

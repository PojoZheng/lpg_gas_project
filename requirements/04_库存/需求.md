# 库存模块需求文档

## 1. 模块定位

**一句话描述**：管理空瓶、重瓶库存，记录进出库，预警过期气瓶。

**核心价值**：
- 清楚知道自己还有多少瓶
- 防止气瓶过期（过期不能卖）
- 库存不足时提醒补货

---

## 2. 入口（首页无入口）

**产品经理说明**：库存是低频管理功能，首页不放入口。配送员开单时如果库存不足会提示，需要管理时从「我的」进入。

| 入口 | 来源 | 说明 | 使用场景 |
|-----|------|------|---------|
| **我的-库存管理** | 我的页面 | 管理全部库存、入库、盘点 | 需要详细管理时 |
| **开单时校验** | 创建订单 | 库存不足提示，可跳转补货 | 防止超卖 |

**首页无库存入口**：首页不显示库存状态，只在开单时校验并提示

---

## 3. 库存模型

### 3.1 库存类型

| 类型 | 说明 | 来源 | 去向 |
|-----|------|------|------|
| **重瓶** | 充满液化气的气瓶 | 进货、充装 | 配送卖出 |
| **空瓶** | 用完的空瓶 | 客户退回 | 送去充装 |

### 3.2 库存状态

| 状态 | 说明 |
|-----|------|
| 可售 | 正常可配送的重瓶 |
| 锁定 | 已开单待配送，锁定中 |
| 待检 | 过期需送检的气瓶 |
| 报废 | 损坏不能用的气瓶 |

### 4.2 库存预警

**预警规则（可配置）**：
| 预警类型 | 触发条件 | 提醒方式 |
|---------|---------|---------|
| 重瓶不足 | 库存 < 安全库存 | 首页显示、推送 |
| 空瓶不足 | 库存 < 安全库存 | 首页显示 |
| 即将过期 | 到期前30天 | 库存页红点 |
| 已过期 | 超过检验日期 | 🔴 红色警告 |

**安全库存设置**：
```
15kg重瓶：少于 5瓶 预警
10kg重瓶：少于 3瓶 预警
50kg重瓶：少于 2瓶 预警
```

## 5. 库存操作

### 5.1 采购入库

**场景**：从供应商直接采购新瓶（重瓶）

**操作流程**：
```
点击[采购入库]
    ↓
选择规格：15kg
    ↓
输入数量：10
    ↓
扫描气瓶条码（根据平台配置）
    ↓
输入单价：¥120/瓶
    ↓
选择供应商（选填）
    ↓
确认入库
    ↓
创建采购记录
    ↓
重瓶库存 +10
```

### 5.2 气瓶充装

**场景**：空瓶送去充装站充装后取回

**操作流程**：
```
点击[气瓶充装]
    ↓
【第一步】扫空瓶出库
扫描空瓶条码（根据平台配置）
    ↓
空瓶库存 -N
    ↓
【第二步】填写充装信息
- 规格：15kg
- 充装站：XX充装站
- 充装气量：75kg（5瓶×15kg）
- 单价：¥85/瓶
- 费用小计：¥425
    ↓
【第三步】扫重瓶入库
扫描充装后的重瓶条码（根据平台配置）
    ↓
重瓶库存 +N
    ↓
创建充装记录（完成）
```

**说明**：
- 三个步骤在一个页面连续完成
- 扫码步骤根据平台配置可跳过
- 跳过扫码时不记录具体气瓶，只记数量

### 5.3 库存盘点

**场景**：实际清点发现库存和系统不一致

**操作流程**：
```
点击[库存盘点]
    ↓
15kg重瓶：
系统显示：12瓶
实际数量：[  11  ]瓶
差异原因：[盘点损耗/其他]
    ↓
确认调整
    ↓
系统库存调整为11瓶
生成盘点记录
```

## 6. 过期气瓶管理

### 6.1 气瓶档案

每个气瓶（如扫码管理）记录：
- 气瓶编号
- 规格
- 生产日期
- 上次检验日期
- 下次检验日期
- 当前位置（在库/在客户处）

### 6.2 过期提醒

**提醒时间线**：
```
到期前30天：🟡 黄色提醒，准备送检
到期前7天：  🟠 橙色提醒，尽快送检
已过期：     🔴 红色警告，不能配送
```

**处理流程**：
```
过期气瓶
    ↓
标记为"待检"
    ↓
送去检验站
    ↓
检验合格：更新日期，重新可用
检验不合格：标记报废
```

## 8. 与订单的关联

### 8.1 开单时库存校验

```
创建订单 15kg×1
    ↓
检查15kg重瓶库存
    ↓
库存充足（>0）→ 允许开单，锁定库存
库存不足（=0）→ 提示"库存不足，请先进货"
```

### 8.2 订单完成时库存变动

**稍后配送订单**：
- 开单时：重瓶库存锁定（-1，状态：锁定）
- 送达后：锁定→出库（正式-1）
- 收回空瓶：空瓶库存+1

**当场完成订单**：
- 完成时：重瓶库存直接-1
- 收回空瓶：空瓶库存+1

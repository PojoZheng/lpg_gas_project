## 2. 库存首页

### 2.1 页面入口（首页无入口）

**入口点**：
1. **我的页面入口**：我的 → 库存管理（主要入口）
2. **开单时校验**：库存不足提示，可跳转补货

**首页无库存入口** - 首页不显示库存状态，只在开单时校验

**产品经理视角**：
- 库存是低频管理功能，首页不放入口
- 配送员开单时如果库存不足会提示
- 需要管理时才从「我的」进入

**配送员视角**：
- 正常开单，库存不足时系统自动提示
- 需要盘点或查明细时去「我的-库存管理」

### 3.2 气瓶状态（仅用于追溯，配送员不常看）

> **产品经理说明**：气瓶列表是给「对账/检查/追溯」用的，不是日常功能
> **配送员使用场景**：客户说气瓶有问题，要查这瓶哪买的、什么时候的

| 标识 | 状态 | 说明 |

**搜索**：
- 支持气瓶编号模糊搜索
- 支持扫码搜索

**筛选条件**：
- 规格：15kg/10kg/50kg
- 状态：重瓶/空瓶/锁定/超期/过期
- 时间范围：入库时间

## 8. 核心交互

### 8.1 库存锁定/解锁

**锁定（创建稍后配送订单时）**：
```
确认开单（稍后配送）
    ↓
检查库存
    ↓
库存充足 → 锁定库存（状态: 锁定）
    ↓
订单关联气瓶（可选）
    ↓
加入待送列表
```

**解锁（订单完成/取消时）**：
```
订单完成
    ↓
扣减库存（正式出库）
    ↓
或订单取消/修改
    ↓
解锁库存（恢复可用）
```

### 8.2 库存校验

**开单时校验**：
```javascript
function checkInventory(spec, quantity, orderType) {
  const available = getAvailableInventory(spec);
  
  if (orderType === 'LATER') {
    // 稍后配送：检查可用库存
    if (available < quantity) {
      return { ok: false, msg: '库存不足，当前可用' + available + '瓶' };
    }
  } else {
    // 当场完成：检查实际库存
    const actual = getActualInventory(spec);
    if (actual < quantity) {
      return { ok: false, msg: '库存不足，当前库房' + actual + '瓶' };
    }
  }
  
  return { ok: true };
}
```

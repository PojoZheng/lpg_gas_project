## 📱 产品概述

### 目标用户

**配送员**（代替原"气贩子"称呼）：
- 个体燃气配送员
- 小型燃气配送企业
- 日送20-50瓶

### 核心场景

| 场景 | 频率 | 描述 | 入口 |
|-----|------|------|------|
| **快速开单** | 极高频 | 2步完成订单创建，支持连续开单 | 首页-快速开单 |
| **待送列表** | 高频 | 查看今日配送任务，导航到客户 | 首页-待送卡片 |
| **扫码收款** | 高频 | 微信/支付宝扫码，记录收款 | 订单完成时 |
| **欠瓶处理** | 中频 | 处理客户无空瓶的情况 | 订单完成时选择 |
| **每日对账** | 每日 | 核对当日收入支出，确认利润 | 首页收入卡片 |

### 关键设计

```
┌────────────────────────────────────────────────────────┐
│  核心设计原则：简单、快速、可靠                         │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📱 导航结构（仅3个底部Tab）                            │
│  ┌─────────┬─────────┬─────────┐                       │
│  │  🏠首页  │  👥客户 │  👤我的 │                       │
│  └─────────┴─────────┴─────────┘                       │
│                                                        │
│  • 首页 = 工作台：待送订单、快速开单、收入概览          │
│  • 客户 = 客户管理：列表、详情、新增                    │
│  • 我的 = 个人中心：设置、历史记录、深度功能            │
│                                                        │
│  📋 订单状态（只有2个）                                │
│  • 待配送 → 已完成（没有"配送中"）                      │
│                                                        │
│  🔄 欠瓶处理（订单完成时）                              │
│  • 客户无空瓶 → 选择"欠着" → 下次配送收回              │
│                                                        │
│  📊 财务查看（首页即见）                                │
│  • 今日已收/待收 → 点击看明细                          │
│                                                        │
│  🔒 安检流程（自动弹出）                                │
│  • 订单完成 → 自动弹出安检 → 拍照10秒完成               │
│                                                        │
│  📴 离线优先（无网可用）                                │
│  • 开单、修改、查看都支持离线                           │
│  • 有网后自动同步                                       │
│                                                        │
└────────────────────────────────────────────────────────┘
```

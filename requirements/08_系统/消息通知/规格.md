### 2.2 消息卡片样式

| 状态 | 左侧标识 | 标题颜色 | 背景色 |
|-----|---------|---------|--------|
| 未读（P0紧急） | 🔴 红点 | 黑色粗体 | 白色 |
| 未读（普通） | 🔵 蓝点 | 黑色 | 白色 |
| 已读 | ⚪ 无 | 灰色 | #F5F5F5 |

### 2.3 交互操作

| 操作 | 触发 | 效果 |
|-----|------|------|
| 查看详情 | 点击卡片 | 进入消息详情或跳转关联页面 |
| 标记已读 | 左滑卡片 | 显示"已读"按钮，点击后变灰 |
| 删除消息 | 长按卡片 | 弹出菜单：删除/标记已读 |
| 全部已读 | 点击右上角 | 所有未读变已读，Toast提示 |
| 快捷操作 | 点击卡片底部按钮 | 跳转到对应功能页面 |

## 3. Tab切换

### 3.1 Tab定义

| Tab | 显示内容 | 排序 |
|-----|---------|------|
| 全部 | 所有消息 | 时间倒序 |
| 未读 | 未读消息 | 优先级>时间 |
| 业务 | 业务提醒（欠瓶/库存/日结） | 时间倒序 |
| 系统 | 系统通知（版本/政策） | 时间倒序 |

### 3.2 切换动画

```
点击Tab
    ↓
当前列表淡出（150ms）
    ↓
加载新数据（骨架屏/loading）
    ↓
新列表淡入（200ms）
    ↓
Tab下划线滑动动画（200ms，ease-out）
```

## 7. 红点/角标机制

### 7.1 首页红点

| 位置 | 显示条件 | 样式 |
|-----|---------|------|
| 收入卡片 | 有待收金额 | 右上角小红点 |
| 待送列表 | 有欠瓶客户 | 客户卡片上"⚠️ 欠X瓶" |

### 7.2 我的页面红点

| 位置 | 显示条件 | 样式 |
|-----|---------|------|
| 消息中心 | 有未读消息 | 右数字角标 |
| 库存管理 | 库存预警 | 右红点 |
| 债务管理 | 有超期欠瓶/欠款 | 右红点 |

### 7.3 App图标角标

```
未读消息数 > 0 时
    ↓
显示数字角标（iOS）
或红点（Android，部分机型）
    ↓
点击App进入后
    ↓
首页加载完成后清除（如消息已读）
或保留（如仍有未读）
```

## 8. 动画效果

### 8.1 列表动画

| 动画 | 触发 | 效果 | 时长 |
|-----|------|------|------|
| 新消息进入 | 收到推送 | 顶部插入，高度从0展开 | 300ms |
| 标记已读 | 点击已读 | 颜色渐变变灰，蓝点消失 | 200ms |
| 删除消息 | 左滑删除 | 向左滑出，高度收缩 | 300ms |
| 全部已读 | 点击按钮 | 所有红点同时消失 | 200ms |

### 8.2 推送到达动画

```
App在前台：
收到推送 → 顶部显示横幅通知
               ↓ 从顶部滑入（300ms）
           停留3秒或点击
               ↓
           向上滑出消失（200ms）
           或点击进入详情

App在后台：
系统推送 → 锁屏/通知栏显示
           点击后唤起App
           进入对应页面
```

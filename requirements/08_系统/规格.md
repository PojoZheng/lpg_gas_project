## 1. 页面清单

| 页面 | 路径 | 说明 |
|-----|------|------|
| 启动页 | `/splash` | App启动展示 |
| 登录页 | `/login` | 手机号/验证码登录 |

| 个人中心 | `/profile` | 我的页面 |
| 个人资料 | `/profile/edit` | 编辑个人资料 |
| 消息中心 | `/notifications` | 系统消息列表 |
| 消息详情 | `/notifications/:id` | 单条消息详情 |
| 设置页 | `/settings` | 系统设置 |
| 数据同步页 | `/sync` | 手动同步页面 |
| 关于页 | `/about` | 关于应用 |

**启动流程**：
```

    ↓

    ↓
未登录/过期 ──► 进入登录页
```

**登录流程**：
```
输入手机号（11位）
    ↓
点击[获取验证码]
    ↓
校验手机号格式
    ↓
发送验证码请求
    ↓
开始60秒倒计时
    ↓
输入6位验证码
    ↓
点击[登录]
    ↓
验证中...（loading）
    ↓
┌─────────────────────────────────────┐
│ 验证成功                            │
│                                     │
│ 检查用户是否存在：                   │
│ ├─ 新用户 → 进入【完善信息页】       │
│ └─ 老用户 → 进入首页                 │
└─────────────────────────────────────┘
```

**验证码规则**：
| 规则 | 说明 |
|-----|------|
| 有效期 | 5分钟 |
| 获取间隔 | 60秒 |
| 每日上限 | 10条/手机号 |
| 错误次数 | 最多5次，超限需重新获取 |
| 格式 | 6位数字 |

**异常情况处理**：
```
手机号格式错误 → Toast "请输入正确的手机号"
验证码错误 → Toast "验证码错误，请重新输入"
网络错误 → Dialog "网络异常，请检查网络后重试"
```

### 2.4 登录态管理

**Token机制**：
```
登录成功
    ↓
下发AccessToken（2小时有效）
下发RefreshToken（30天有效）
    ↓
本地加密存储
    ↓
每次请求携带AccessToken
    ↓
AccessToken过期 → 用RefreshToken刷新
    ↓
RefreshToken过期 → 重新登录
```

### 4.4 消息推送机制

**本地推送**（App运行时）：
```
触发条件满足
    ↓
检查推送设置（是否开启该类型）
    ↓
检查免打扰时段
    ↓
显示In-App通知横幅
    ↓
点击跳转对应页面
```

**远程推送**（App后台/关闭时）：
```
服务端触发
    ↓
推送服务（APNs/FCM）
    ↓
设备接收
    ↓
点击通知
    ↓
唤起App并跳转对应页面
```

## 8. 核心交互

### 8.1 启动流程

```
点击App图标
    ↓
显示启动页（2秒，可跳过）
    ↓
检查版本更新
    ↓
强制更新 → 跳转应用商店
    ↓
可选更新 → 提示，可选择跳过
    ↓
检查登录态
    ↓
未登录 → 登录页
    ↓
已登录 → 静默同步 → 进入首页
```

### 8.2 后台切前台

```
从后台切回前台
    ↓
检查登录态（超过24小时需重新验证）
    ↓
检查网络状态
    ↓
网络恢复 → 自动同步
    ↓
有新数据 → 显示提示"数据已更新"
```

### 8.3 退出登录

```
点击[退出登录]
    ↓
确认弹窗：
┌─────────────────────────────────────────┐
│  确认退出登录？                          │
│                                         │
│  退出后需要重新验证手机号才能使用。      │
│                                         │
│  [取消]              [确认退出]          │
└─────────────────────────────────────────┘
    ↓
清除本地登录态
    ↓
保留本地数据（方便下次登录）
    ↓
进入登录页
```

# 平台端配置规格文档

## 1. 页面清单

| 页面 | 路径 | 说明 |
|-----|------|------|
| 地区模板管理 | `/admin/regions` | 管理各地区合规模板 |
| 模板详情配置 | `/admin/regions/:id` | 单个地区详细配置 |
| 安检规则配置 | `/admin/inspection-rules` | 全局安检规则设置 |
| 扫码配置 | `/admin/scan-config` | 气瓶扫码溯源配置 |
| 数据上报配置 | `/admin/report-config` | 政府监管上报配置 |
| 版本管理 | `/admin/versions` | App版本发布管理 |
| 运营商管理 | `/admin/operators` | 燃气企业接入管理 |

## 7. 配置生效机制

### 7.1 配置下发流程

```
平台端修改配置
    ↓
保存并提交审核（重要配置）
    ↓
审核通过
    ↓
配置写入配置中心
    ↓
实时推送至客户端（在线用户）
    ↓
或客户端轮询拉取（默认5分钟）
    ↓
客户端缓存并生效
```

### 7.2 配置优先级

| 优先级 | 配置来源 | 说明 |
|-------|---------|------|
| 1 | 地区特定配置 | 针对具体城市的配置 |
| 2 | 模板默认配置 | 严格/标准/宽松模板的默认值 |
| 3 | 全局默认配置 | 系统硬编码的默认值 |

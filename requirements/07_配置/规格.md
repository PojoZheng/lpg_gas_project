# 配置模块规格文档

## 1. 页面清单

| 页面 | 路径 | 说明 |
|-----|------|------|
| 业务规则页 | `/settings/business` | 配送员自主配置 |
| 价格配置页 | `/settings/pricing` | 押金/租金/残液价格 |
| 预警配置页 | `/settings/warning` | 库存/欠瓶/欠款预警线 |
| 打印配置页 | `/settings/print` | 小票打印设置 |
| 数据管理页 | `/settings/data` | 数据备份/恢复 |
| 关于页面 | `/settings/about` | 版本/帮助/反馈 |

## 7. 配置生效机制

### 7.1 配置存储

```
配送端配置
    ↓
本地存储（SQLite）
    ↓
实时生效（新订单使用新配置）
    ↓
云端同步（后台自动同步）
```

### 7.2 配置优先级

| 优先级 | 配置来源 | 说明 |
|-------|---------|------|
| 1 | 用户自定义 | 配送员自己设置的 |
| 2 | 平台默认 | 平台提供的默认值 |
| 3 | 系统硬编码 | 最后一道防线 |

### 7.3 配置变更通知

```
配置变更
    ↓
本地保存
    ↓
触发配置变更事件
    ↓
相关模块刷新：
- 订单模块（押金/租金计算）
- 库存模块（预警线）
- 财务模块（残液处理）
- 债务模块（提醒天数）
```

# 配置模块数据模型

## 1. 实体关系图

```
┌───────────────────┐
│  DealerConfig    │
│  配送员配置       │
└───────────────────┘
         │
         │ 1:1
         ▼
┌───────────────────┐
│   AppSettings    │
│   应用设置        │
└───────────────────┘
```

---

## 2. DealerConfig 配送员业务配置

### 2.1 表结构

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | STRING(32) | 是 | UUID | 主键 |
| dealer_id | STRING(32) | 是 | - | 配送员ID，唯一索引 |
| 
| -- 押金配置 -- | | | | |
| deposit_15kg | DECIMAL(10,2) | 是 | 150.00 | 15kg押金 |
| deposit_10kg | DECIMAL(10,2) | 是 | 100.00 | 10kg押金 |
| deposit_50kg | DECIMAL(10,2) | 是 | 400.00 | 50kg押金 |
| 
| -- 租金配置 -- | | | | |
| rent_monthly | DECIMAL(10,2) | 是 | 5.00 | 月租金 |
| rent_yearly | DECIMAL(10,2) | 是 | 50.00 | 年租金 |
| 
| -- 残液配置 -- | | | | |
| residual_price | DECIMAL(10,2) | 是 | 5.00 | 残液单价(元/kg) |
| residual_default | ENUM | 是 | DEDUCT | 默认处理方式 |
| 
| -- 欠瓶配置 -- | | | | |
| owed_remind_days | JSON | 是 | [7,15,30] | 提醒天数数组 |
| owed_max_days | INTEGER | 是 | 60 | 最长欠瓶天数 |
| owed_overdue_action | ENUM | 是 | CONVERT | 超期处理方式 |
| 
| -- 欠款配置 -- | | | | |
| debt_max_amount | DECIMAL(10,2) | 是 | 1000.00 | 最大欠款额度 |
| debt_max_days | INTEGER | 是 | 30 | 最长欠款天数 |
| debt_remind_days | JSON | 是 | [7,15,25] | 提醒天数数组 |
| 
| -- 库存预警配置 -- | | | | |
| warning_15kg | INTEGER | 是 | 5 | 15kg预警线 |
| warning_10kg | INTEGER | 是 | 3 | 10kg预警线 |
| warning_50kg | INTEGER | 是 | 2 | 50kg预警线 |
| warning_lock_days | INTEGER | 是 | 3 | 超期锁定预警天数 |
| warning_expire_days | INTEGER | 是 | 30 | 过期提醒天数 |
| 
| -- 其他配置 -- | | | | |
| is_default | BOOLEAN | 是 | FALSE | 是否使用默认配置 |
| created_at | DATETIME | 是 | CURRENT | 创建时间 |
| updated_at | DATETIME | 是 | CURRENT | 更新时间 |

### 2.2 枚举定义

**残液处理方式 (residual_default)**：
```
DEDUCT          = '抵扣气款'
SEPARATE        = '单独收款'
IGNORE          = '不收取'
```

**超期处理方式 (owed_overdue_action)**：
```
CONVERT         = '转为收入'
CONTINUE        = '继续欠着'
```

### 2.3 默认配置值

```json
{
  "deposit_15kg": 150.00,
  "deposit_10kg": 100.00,
  "deposit_50kg": 400.00,
  "rent_monthly": 5.00,
  "rent_yearly": 50.00,
  "residual_price": 5.00,
  "residual_default": "DEDUCT",
  "owed_remind_days": [7, 15, 30],
  "owed_max_days": 60,
  "owed_overdue_action": "CONVERT",
  "debt_max_amount": 1000.00,
  "debt_max_days": 30,
  "debt_remind_days": [7, 15, 25],
  "warning_15kg": 5,
  "warning_10kg": 3,
  "warning_50kg": 2,
  "warning_lock_days": 3,
  "warning_expire_days": 30
}
```

## 3. AppSettings 应用设置

### 3.1 表结构

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | STRING(32) | 是 | UUID | 主键 |
| dealer_id | STRING(32) | 是 | - | 配送员ID，唯一索引 |
| 
| -- 推送设置 -- | | | | |
| push_enabled | BOOLEAN | 是 | TRUE | 开启推送 |
| push_start_time | TIME | 是 | 08:00 | 推送开始时间 |
| push_end_time | TIME | 是 | 20:00 | 推送结束时间 |
| push_inventory | BOOLEAN | 是 | TRUE | 库存不足提醒 |
| push_owed_bottle | BOOLEAN | 是 | TRUE | 欠瓶超期提醒 |
| push_owed_money | BOOLEAN | 是 | TRUE | 欠款超期提醒 |
| push_settlement | BOOLEAN | 是 | TRUE | 日结对账提醒 |
| push_inspection | BOOLEAN | 是 | TRUE | 定期安检提醒 |
| 
| -- 打印设置 -- | | | | |
| print_enabled | BOOLEAN | 是 | TRUE | 开启打印 |
| print_auto | BOOLEAN | 是 | TRUE | 自动打印 |
| print_copies | INTEGER | 是 | 1 | 打印份数 |
| print_device_id | STRING(50) | 否 | NULL | 打印机设备ID |
| print_device_name | STRING(100) | 否 | NULL | 打印机名称 |
| print_content | JSON | 是 | {...} | 打印内容配置 |
| 
| -- 显示设置 -- | | | | |
| theme | ENUM | 是 | AUTO | 主题 |
| font_size | ENUM | 是 | MEDIUM | 字体大小 |
| 
| -- 数据设置 -- | | | | |
| auto_backup | BOOLEAN | 是 | TRUE | 自动备份 |
| backup_frequency | ENUM | 是 | DAILY | 备份频率 |
| last_backup_at | DATETIME | 否 | NULL | 上次备份时间 |
| 
| created_at | DATETIME | 是 | CURRENT | 创建时间 |
| updated_at | DATETIME | 是 | CURRENT | 更新时间 |

### 3.2 枚举定义

**主题 (theme)**：
```
LIGHT           = '浅色'
DARK            = '深色'
AUTO            = '跟随系统'
```

**字体大小 (font_size)**：
```
SMALL           = '小'
MEDIUM          = '中'
LARGE           = '大'
```

**备份频率 (backup_frequency)**：
```
DAILY           = '每天'
WEEKLY          = '每周'
MONTHLY         = '每月'
```

### 3.3 打印内容配置

```json
{
  "print_content": {
    "show_dealer_info": true,
    "show_customer_info": true,
    "show_order_detail": true,
    "show_amount": true,
    "show_qrcode": true,
    "show_ad": false,
    "ad_content": ""
  }
}
```

## 4. PlatformConfig 平台配置

### 4.1 表结构（平台端维护）

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | STRING(32) | 是 | UUID | 主键 |
| region_code | STRING(20) | 是 | - | 地区编码 |
| region_name | STRING(50) | 是 | - | 地区名称 |
| 
| -- 安检配置 -- | | | | |
| inspection_required | BOOLEAN | 是 | TRUE | 是否强制安检 |
| inspection_mode | ENUM | 是 | SUGGEST | 安检模式 |
| inspection_cycle | INTEGER | 是 | 90 | 定期安检周期(天) |
| ai_recognition | BOOLEAN | 是 | TRUE | AI识别开关 |
| required_photos | JSON | 是 | ["cylinder"] | 必拍项目 |
| 
| -- 扫码配置 -- | | | | |
| scan_required_purchase | BOOLEAN | 是 | FALSE | 采购入库是否必须扫码 |
| scan_required_refill | BOOLEAN | 是 | FALSE | 充装是否必须扫码 |
| scan_required_delivery | BOOLEAN | 是 | FALSE | 配送出库是否必须扫码 |
| 
| -- 上报配置 -- | | | | |
| report_mode | ENUM | 是 | ENTERPRISE | 上报模式 |
| report_frequency | ENUM | 是 | DAILY | 上报频率 |
| report_retry_count | INTEGER | 是 | 3 | 失败重试次数 |
| 
| -- 价格指导 -- | | | | |
| guide_price_15kg | DECIMAL(10,2) | 否 | NULL | 15kg指导价 |
| guide_price_10kg | DECIMAL(10,2) | 否 | NULL | 10kg指导价 |
| guide_price_50kg | DECIMAL(10,2) | 否 | NULL | 50kg指导价 |
| 
| status | ENUM | 是 | ACTIVE | 状态 |
| created_at | DATETIME | 是 | CURRENT | 创建时间 |
| updated_at | DATETIME | 是 | CURRENT | 更新时间 |

### 4.2 枚举定义

**安检模式 (inspection_mode)**：
```
REQUIRED        = '强制'
SUGGEST         = '建议'
OPTIONAL        = '可选'
```

**上报模式 (report_mode)**：
```
ENTERPRISE      = '燃企中转'
DIRECT          = '直连政府'
MIXED           = '混合模式'
```

**上报频率 (report_frequency)**：
```
REALTIME        = '实时'
HOURLY          = '每小时'
DAILY           = '每天'
```

## 5. 配置同步机制

### 5.1 同步策略

```
配送端修改配置
    ↓
本地立即生效
    ↓
后台异步同步到云端
    ↓
其他设备登录时拉取最新配置
```

### 5.2 同步字段

| 配置表 | 同步方向 | 冲突处理 |
|-------|---------|---------|
| DealerConfig | 双向 | 时间戳优先 |
| AppSettings | 双向 | 设备本地优先 |
| PlatformConfig | 云端→本地 | 云端优先 |

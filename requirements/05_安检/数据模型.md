# 安检模块数据模型

## 1. 实体关系图

```
┌───────────────────┐
│InspectionRecord   │
│    安检记录        │
└─────────┬─────────┘
          │ 1:N
          ▼
┌───────────────────┐
│InspectionItem     │
│    检查项明细      │
└─────────┬─────────┘
          │ 1:N
          ▼
┌───────────────────┐     ┌───────────────────┐
│InspectionPhoto    │     │   DangerRecord    │
│    安检照片        │     │     隐患记录       │
└───────────────────┘     └───────────────────┘
                                    │
                                    │ 1:N
                                    ▼
                          ┌───────────────────┐
                          │DangerHandleRecord │
                          │   隐患处理记录     │
                          └───────────────────┘
```

## 2. InspectionRecord 安检记录

### 2.1 表结构

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | STRING(32) | 是 | UUID | 主键 |
| dealer_id | STRING(32) | 是 | - | 配送员ID |
| customer_id | STRING(32) | 是 | - | 客户ID |
| order_id | STRING(32) | 否 | NULL | 关联订单ID |
| inspection_type | ENUM | 是 | DELIVERY | 安检类型 |
| inspection_date | DATE | 是 | CURRENT | 安检日期 |
| inspection_time | TIME | 是 | CURRENT | 安检时间 |
| overall_result | ENUM | 是 | NORMAL | 综合结果 |
| has_danger | BOOLEAN | 是 | FALSE | 是否有隐患 |
| danger_level | ENUM | 否 | NULL | 隐患等级 |
| danger_desc | STRING(500) | 否 | NULL | 隐患描述 |
| cylinder_count | INTEGER | 是 | 1 | 检查气瓶数 |
| photo_count | INTEGER | 是 | 0 | 照片数量 |
| sync_status | ENUM | 是 | PENDING | 同步状态 |
| sync_time | DATETIME | 否 | NULL | 同步时间 |
| sync_response | JSON | 否 | NULL | 同步响应 |
| gov_report_id | STRING(50) | 否 | NULL | 政府上报ID |
| longitude | DECIMAL(10,7) | 否 | NULL | 经度 |
| latitude | DECIMAL(10,7) | 否 | NULL | 纬度 |
| remark | STRING(200) | 否 | NULL | 备注 |
| created_at | DATETIME | 是 | CURRENT | 创建时间 |
| updated_at | DATETIME | 是 | CURRENT | 更新时间 |

### 2.2 枚举定义

**安检类型 (inspection_type)**：
```
DELIVERY        = '随瓶安检'      // 配送时进行
PERIODIC        = '定期安检'      // 按周期进行
SUPPLEMENT      = '补录安检'      // 事后补录
```

**综合结果 (overall_result)**：
```
NORMAL          = '正常'
ABNORMAL        = '异常'
SKIPPED         = '跳过'
```

**隐患等级 (danger_level)**：
```
NORMAL          = '一般隐患'
MAJOR           = '重大隐患'
```

**同步状态 (sync_status)**：
```
PENDING         = '待同步'
SYNCING         = '同步中'
SYNCED          = '已同步'
FAILED          = '同步失败'
```

## 8. 视图/查询

### 8.1 待同步安检记录查询

```sql
-- 待同步的安检记录
SELECT *
FROM inspection_record
WHERE dealer_id = ?
  AND sync_status IN ('PENDING', 'FAILED')
ORDER BY created_at ASC
LIMIT 50
```

### 8.2 定期安检任务查询

```sql
-- 生成定期安检任务
SELECT 
  c.id as customer_id,
  c.name as customer_name,
  MAX(ir.inspection_date) as last_inspection_date,
  DATE_ADD(MAX(ir.inspection_date), INTERVAL 90 DAY) as next_due_date,
  DATEDIFF(CURRENT_DATE, DATE_ADD(MAX(ir.inspection_date), INTERVAL 90 DAY)) as days_overdue
FROM customer c
LEFT JOIN inspection_record ir ON c.id = ir.customer_id
WHERE c.dealer_id = ?
  AND c.status = 'ACTIVE'
GROUP BY c.id
HAVING next_due_date <= DATE_ADD(CURRENT_DATE, INTERVAL 15 DAY)
   OR last_inspection_date IS NULL
ORDER BY next_due_date ASC
```

### 8.3 隐患统计查询

```sql
-- 隐患统计
SELECT 
  danger_type,
  danger_level,
  COUNT(*) as count,
  SUM(CASE WHEN status = 'COMPLETED' THEN 1 ELSE 0 END) as resolved_count
FROM danger_record
WHERE dealer_id = ?
  AND created_at >= DATE_SUB(CURRENT_DATE, INTERVAL 30 DAY)
GROUP BY danger_type, danger_level
```

## 9. 索引设计

| 表名 | 索引名 | 字段 | 类型 |
|-----|-------|------|------|
| inspection_record | idx_dealer_date | dealer_id, inspection_date | 普通索引 |
| inspection_record | idx_order | order_id | 普通索引 |
| inspection_record | idx_customer | customer_id | 普通索引 |
| inspection_record | idx_sync_status | dealer_id, sync_status | 普通索引 |
| inspection_item | idx_inspection | inspection_id | 普通索引 |
| inspection_photo | idx_inspection | inspection_id | 普通索引 |
| danger_record | idx_dealer_status | dealer_id, status | 普通索引 |
| danger_record | idx_inspection | inspection_id | 唯一索引 |
| periodic_inspection_task | idx_dealer_due | dealer_id, next_due_date | 普通索引 |
| periodic_inspection_task | idx_customer | customer_id | 普通索引 |

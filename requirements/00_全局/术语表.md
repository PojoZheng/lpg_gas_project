# 术语表

本文档统一项目中使用的专业术语，避免歧义。

---

## 角色术语

| 术语 | 英文 | 说明 |
|-----|------|------|
| 配送员 | Dealer/Delivery Person | 个体液化气配送员，本系统的核心用户（旧称"气贩子"） |
| 平台方 | Platform | 系统的运营方 |
| 客户 | Customer | 终端用户，使用液化气的居民或商户 |
| 燃企 | Gas Company | 液化气供应商/充装站 |

---

## 业务术语

| 术语 | 说明 |
|-----|------|
| 重瓶 | 充满液化气的气瓶，可配送 |
| 空瓶 | 用完液化气的气瓶，需回收 |
| 欠瓶 | 送重瓶后未收回空瓶的状态 |
| 残液 | 空瓶中剩余的液化气 |
| 押金 | 客户租用气瓶的保证金 |
| 租金 | 气瓶的月租/年租费用 |
| 气款 | 液化气的费用 |
| **采购** | 从供应商直接购买新瓶（重瓶），增加库存 |
| **充装** | 将空瓶送到充装站充装后取回，空瓶减、重瓶增 |
| 随瓶安检 | 配送时同步进行的安全检查 |
| 定期安检 | 按规定周期（如90天）进行的安检 |
| 电子签名 | 客户确认订单/欠瓶的数字化签名（可配置） |

---

## 订单状态

| 状态 | 说明 |
|-----|------|
| 草稿 | 临时保存，未确认 |
| 待配送 | 已确认，等待配送（系统只有待配送和已完成两个状态） |
| 已完成 | 配送完成，已收款 |
| 已取消 | 订单取消 |
| 已退货 | 客户退货 |
| 已换货 | 客户换货 |

> **注意**：本系统没有"配送中"状态，配送员配送过程不需要在App中标记状态。

---

## 欠瓶状态

| 状态 | 说明 |
|-----|------|
| 未收回 | 欠瓶尚未收回 |
| 已收回 | 欠瓶已收回 |
| 转为收入 | 用押金抵扣转为收入 |
| 转为欠款 | 转为客户的欠款 |
| 作废 | 气贩子承担损失 |

## 技术术语

| 术语 | 说明 |
|-----|------|
| AI安检 | 使用AI图像识别辅助安全检查 |
| 数据上报 | 将配送、安检数据上报给政府监管平台 |
| 气瓶扫码 | 通过二维码/RFID识别气瓶身份 |
| 驾驶模式 | 配送过程中的简化操作界面 |
| 地图模式 | 基于地图的配送管理界面 |
| 离线优先 | 无网络时也能正常使用，有网后自动同步 |
| 连续开单 | 完成一单后不关闭面板，继续开下一单 |
| 5秒撤销 | 关键操作后5秒内可撤销 |
| 24小时修改 | 订单完成后24小时内可修改 |

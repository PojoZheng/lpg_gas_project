## 待补充实体详细说明

### 库存域（必须补充）

**Inventory（库存表）**
```
- id: 库存ID
- dealer_id: 气贩子ID
- cylinder_spec: 规格（15kg/10kg/50kg）
- type: 类型（full重瓶/empty空瓶）
- quantity: 数量
- alert_threshold: 预警阈值
- created_at
- updated_at
```

**InventoryLog（库存变动日志）**
```
- id: 日志ID
- dealer_id: 气贩子ID
- cylinder_spec: 规格
- type: 变动类型（in入库/out出库/lock锁定/unlock解锁）
- quantity: 变动数量
- related_order_id: 关联订单ID
- related_type: 关联类型（order/inspection/refill）
- remark: 备注
- created_at
```

### 财务域（必须补充）

**IncomeRecord（收入记录）**
```
- id: 记录ID
- dealer_id: 气贩子ID
- order_id: 关联订单ID
- amount: 金额
- type: 类型（gas气款/fee配送费/deposit押金/rent租金/residual残液）
- payment_method: 支付方式
- date: 日期
- created_at
```

**ExpenseRecord（支出记录）**
```
- id: 记录ID
- dealer_id: 气贩子ID
- type: 类型（refill充装/purchase采购/delivery配送费）
- amount: 金额
- cylinder_spec: 规格
- quantity: 数量
- date: 日期
- created_at
```

**DailySettlement（日结对账）**
```
- id: 结算ID
- dealer_id: 气贩子ID
- date: 日期
- total_income: 总收入
- total_expense: 总支出
- profit: 利润
- order_count: 订单数
- details: 明细JSON
- created_at
```

**OwedRecord（欠款记录）**
```
- id: 欠款ID
- dealer_id: 气贩子ID
- customer_id: 客户ID
- order_id: 关联订单ID
- amount: 欠款金额
- status: 状态（outstanding未还/paid已还）
- created_at
- paid_at
```

### 安检域（必须补充）

**Inspection（安检记录）**
```
- id: 安检ID
- order_id: 关联订单ID
- dealer_id: 气贩子ID
- customer_id: 客户ID
- status: 状态
- result: 结果（pass通过/fail不通过）
- ai_detected: AI识别隐患（JSON）
- created_at
```

**InspectionItem（安检项）**
```
- id: 项ID
- inspection_id: 安检ID
- item_type: 检查项类型
- result: 结果（normal正常/hazard隐患）
- photo_url: 照片URL
- remark: 备注
```

### 配置域（必须补充）

**DealerConfig（气贩子配置）**
```
- dealer_id: 气贩子ID
- deposit_config: 押金配置JSON
- rent_config: 租金配置JSON
- residual_config: 残液配置JSON
- credit_config: 欠款配置JSON
- inventory_alert: 库存预警配置JSON
- updated_at
```

**PlatformConfig（平台配置）**
```
- region: 地区
- cylinder_scan: 扫码配置JSON
- safety_inspection: 安检配置JSON
- data_reporting: 上报配置JSON
```

### 系统域（必须补充）

**OfflineQueue（离线操作队列）**
```
- id: 队列ID
- dealer_id: 气贩子ID
- operation_type: 操作类型
- operation_data: 操作数据JSON
- status: 状态（pending待同步/syncing同步中/completed已完成/failed失败）
- retry_count: 重试次数
- created_at
- synced_at
```

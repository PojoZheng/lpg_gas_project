# 任务 PRD：认证接口契约对齐

## 任务目标
将认证相关接口（send-code/login/refresh/devices）统一到 shared 契约，确保前后端字段、错误码、提示语一致。

## 业务背景
当前接口可用，但字段命名与错误表达存在演进空间，若不提前统一会影响后续任务迭代效率。

## 范围边界
### 本任务包含
1. 认证接口响应结构统一
2. 错误码与错误信息统一
3. 前端调用层同步适配
4. 接口说明文档更新

### 本任务不包含
- 新增业务接口
- 页面视觉改版
- 用户体系扩展

## 设计与规范约束
- 中文错误提示
- 兼容现有登录页交互
- 禁用 emoji 文案

## 关联需求
- `requirements/08_系统/登录认证/接口.md`
- `requirements/08_系统/登录认证/需求.md`
- `requirements/08_系统/规格.md`

## 验收标准
- 核心认证接口结构统一
- 前端调用不报错且提示中文
- 自动化测试通过
- 完成记录写入 `completion.md`

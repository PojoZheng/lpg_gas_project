# Task 47：客户域 P1 - 客户列表与详情管理

## 1. 背景与目标

### 1.1 业务背景
- 工作台底部 Sheet 快速开单已支持客户选择
- 但缺乏完整的客户管理入口
- 配送员需要查看客户历史、欠瓶欠款、快速联系

### 1.2 目标
- 提供客户列表页（搜索、筛选、标记）
- 提供客户详情页（信息、台账、历史、快捷操作）
- 支持从客户详情直接开单


#### 2.2.2 功能说明

| 功能 | 说明 |
|------|------|
| 搜索 | 实时搜索（debounce 300ms），匹配姓名/电话/地址 |
| 筛选 | 全部 / 有欠瓶 / 有欠款 / VIP客户 |
| 客户卡片 | 点击进入客户详情 |
| 标记 | 欠瓶/欠款用红色标签，VIP用主色标签 |
| +新客户 | 跳转 quick-order.html?intent=newCustomer |

#### 2.2.3 排序规则

1. 有欠款/欠瓶的优先
2. 最近有订单的其次
3. 按姓名拼音排序

## 3. 接口设计

### 3.1 GET /customers

查询客户列表

**请求参数：**
```json
{
  "keyword": "李",
  "filter": "all|owed_bottle|owed_money|vip",
  "page": 1,
  "size": 20
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "CUST-001",
        "name": "李大妈",
        "phone": "13800138000",
        "address": "阳光花园5栋1单元301",
        "tags": ["VIP"],
        "owedEmptyCount": 2,
        "owedAmount": 150,
        "lastOrderAt": "2026-04-12T10:30:00Z"
      }
    ],
    "pagination": {
      "page": 1,
      "size": 20,
      "total": 15
    }
  }
}
```

### 3.2 GET /customers/:id

客户详情 + 台账汇总

**响应：**
```json
{
  "success": true,
  "data": {
    "id": "CUST-001",
    "name": "李大妈",
    "phone": "13800138000",
    "address": "阳光花园5栋1单元301",
    "tags": ["VIP"],
    "note": "Prefer 15kg",
    "ledger": {
      "owedEmptyCount": 2,
      "owedEmptySpec": "15kg",
      "owedAmount": 150,
      "owedSince": "2026-03-28"
    },
    "stats": {
      "totalOrders": 28,
      "totalAmount": 3360,
      "lastOrderAt": "2026-04-12T10:30:00Z"
    }
  }
}
```

### 3.3 GET /customers/:id/orders

客户历史订单

**请求参数：**
```json
{
  "page": 1,
  "size": 10
}
```

**响应：** 同 order-list 接口，但只返回该客户订单

### 3.4 POST /customers/:id/reminder

记录催款

**请求体：**
```json
{
  "type": "call|sms|wechat",
  "content": "催款话术",
  "result": "promised|unreachable|refused"
}
```

## 4. 页面跳转关系

```
my.html（我的）
    ↓ 点击「客户管理」
customer-list.html（客户列表）
    ├─ 点击客户卡片 ──→ customer-detail.html（客户详情）
    │                       ├─ 快速开单 ──→ quick-order.html?customerId=xxx
    │                       ├─ 查看订单 ──→ order-list.html?customerId=xxx
    │                       └─ 催款 ──────→ 底部催款面板
    │
    └─ 点击[+新客户] ──→ quick-order.html?intent=newCustomer
```

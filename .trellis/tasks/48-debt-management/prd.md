# Task 48：财务域 P1 - 欠款管理与催收

## 1. 背景与目标

### 1.1 业务背景
- 配送过程中存在「记账」情况（客户暂时没钱，先送货后付款）
- 欠瓶和欠款需要管理，防止遗忘
- 需要催款功能，降低坏账风险

### 1.2 目标
- 提供欠款全景视图（总额、逾期、分布）
- 提供催收工具（话术、电话、短信）
- 提供还款记录功能

#### 2.2.2 功能说明

| 功能 | 说明 |
|------|------|
| 统计卡片 | 总欠款金额 + 人数，逾期金额 + 人数（红色标记） |
| 筛选标签 | 全部 / 逾期未还（>7天）/ 今日需催（昨天催过或新逾期） |
| 催收列表 | 按逾期天数倒序，最紧急的在上面 |
| 催款按钮 | 弹出催款面板（话术 + 操作） |
| 记录还款 | 弹出还款录入面板 |

#### 2.2.3 逾期规则

| 逾期天数 | 标记 | 说明 |
|----------|------|------|
| 0-7天 | 🟢 正常 | 新记账，暂未逾期 |
| 8-15天 | 🟡 提醒 | 黄色提醒，建议催款 |
| >15天 | 🔴 逾期 | 红色警告，需重点催收 |

## 3. 接口设计

### 3.1 GET /debts/overview

欠款统计概览

**响应：**
```json
{
  "success": true,
  "data": {
    "totalDebt": {
      "amount": 2340,
      "customerCount": 5
    },
    "overdueDebt": {
      "amount": 1200,
      "customerCount": 3,
      "thresholdDays": 7
    },
    "recentChanges": {
      "newToday": 1,
      "repaidToday": 0
    }
  }
}
```

### 3.2 GET /debts/list

待催收列表

**请求参数：**
```json
{
  "filter": "all|overdue|today",
  "page": 1,
  "size": 20
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "items": [
      {
        "customerId": "CUST-001",
        "customerName": "王老板",
        "phone": "13800138001",
        "debtAmount": 480,
        "debtDays": 15,
        "lastReminderAt": "2026-04-10T14:30:00Z",
        "status": "overdue"
      }
    ]
  }
}
```

### 3.3 POST /debts/reminder

记录催款

**请求体：**
```json
{
  "customerId": "CUST-001",
  "type": "call|sms|wechat|visit",
  "content": "催款话术内容",
  "result": "promised|unreachable|refused|no_answer",
  "promisedAt": "2026-04-15"
}
```

### 3.4 POST /debts/repayment

记录还款

**请求体：**
```json
{
  "customerId": "CUST-001",
  "amount": 480,
  "method": "cash|wechat|alipay|transfer",
  "note": "",
  "fullRepayment": true
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "remainingDebt": 0,
    "repaymentRecord": {
      "id": "REP-001",
      "amount": 480,
      "method": "cash",
      "createdAt": "2026-04-12T16:00:00Z"
    }
  }
}
```

## 4. 数据模型

### 4.1 DebtRecord（欠款记录）

基于客户台账扩展：

```javascript
{
  customerId: "CUST-001",
  customerName: "王老板",
  totalDebt: 480,           // 当前总欠款
  debtHistory: [            // 欠款明细（未还清的订单）
    {
      orderId: "ORD-001",
      amount: 120,
      createdAt: "2026-03-28",
      daysSince: 15
    }
  ],
  repaymentHistory: [       // 还款记录
    {
      id: "REP-001",
      amount: 200,
      method: "cash",
      createdAt: "2026-03-15"
    }
  ],
  reminderHistory: [        // 催款记录
    {
      id: "REM-001",
      type: "call",
      result: "promised",
      promisedAt: "2026-04-15",
      createdAt: "2026-04-10"
    }
  ]
}
```

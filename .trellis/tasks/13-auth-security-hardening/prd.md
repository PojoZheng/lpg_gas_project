# 任务 PRD：认证安全加固

## 任务目标
为认证链路补齐基础安全防护，包括验证码限流、失败次数控制和会话安全校验。

## 业务背景
当前认证流程可用，但缺少生产级安全策略，存在刷码、暴力尝试和会话滥用风险。

## 范围边界
### 本任务包含
1. 验证码发送频次限制
2. 验证码校验失败次数与冷却
3. 刷新令牌与设备下线安全校验
4. 错误码与错误文案统一

### 本任务不包含
- 风控平台接入
- 多因子认证（MFA）
- 复杂行为风控模型

## 设计与规范约束
- 所有错误提示为中文
- 错误码对齐 `shared/api-contracts.md`
- 界面提示不使用 emoji

## 关联需求
- `requirements/08_系统/登录认证/需求.md`
- `requirements/08_系统/规格.md`
- `requirements/00_全局/产品概述.md`

## 验收标准
- 限流和失败次数策略生效
- 关键认证接口具备最小安全校验
- 自动化测试通过
- 完成记录写入 `completion.md`

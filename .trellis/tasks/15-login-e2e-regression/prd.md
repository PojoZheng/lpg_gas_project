# 任务 PRD：登录流程自动化回归

## 任务目标
建立登录链路的自动化回归集，保障后续任务迭代不破坏已完成认证能力。

## 业务背景
当前已有自动触发脚本和基础测试命令，需要将登录流程覆盖扩展为稳定可复跑的回归用例集。

## 范围边界
### 本任务包含
1. 覆盖发送验证码流程
2. 覆盖验证码登录流程
3. 覆盖令牌刷新流程
4. 覆盖设备会话查询/失效流程
5. 测试报告输出和任务追溯

### 本任务不包含
- 非登录业务域（订单/库存/财务）回归
- 平台端 UI 回归
- 性能压测

## 设计与规范约束
- 测试用例说明和断言文本使用中文
- 登录相关页面必须符合中文文案、禁用 emoji、主色一致约束

## 关联需求
- `requirements/08_系统/登录认证/需求.md`
- `requirements/08_系统/登录认证/规格.md`
- `requirements/08_系统/规格.md`

## 验收标准
- 至少 4 条核心回归用例稳定通过
- 自动触发脚本支持 task 级回归命令
- 测试结果可在任务完成记录中追溯

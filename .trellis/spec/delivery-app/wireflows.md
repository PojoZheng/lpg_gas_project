# 关键页面线框流程

## 1. 工作台到快速开单
1. 进入首页，查看今日已收/待收
2. 点击“快速开单”
3. 选择客户
4. 选择规格与数量
5. 选择“稍后配送”或“当场完成”
6. 完成订单创建

## 2. 待配送到完单
1. 从待送列表进入订单详情
2. 确认配送信息
3. 录入实收金额与收款方式
4. 记录空瓶回收或欠瓶
5. 提交完单

## 3. 完单到安检
1. 完单后自动弹出安检面板
2. 默认检查项勾选
3. 上传安检照片
4. 提交安检并异步上报

## 4. 财务日结
1. 进入“我的-财务管理”
2. 查看今日流水汇总
3. 确认日结
4. 生成日结记录

## 5. 视觉与文案约束
- 所有流程页面必须沿用浅色卡片风格。
- 所有按钮与提示文案使用中文。
- 所有页面严禁 emoji。

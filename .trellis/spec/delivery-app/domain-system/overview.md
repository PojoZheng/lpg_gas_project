# 系统域概述

## 目标
提供登录认证、会话管理、消息通知与离线同步基础设施。

## 核心流程
1. 手机号验证码登录并维护会话
2. 管理设备与刷新令牌
3. 承载同步队列与冲突处理入口
4. 展示消息中心与推送设置

## 关键规则
- AccessToken 短期有效，RefreshToken 长周期刷新
- 离线模式下业务可继续，恢复网络后自动同步
- 关键系统消息必须持久化可追溯

## 依赖
- shared 同步协议、后端用户域与通知能力

## MVP 范围
- 登录、会话刷新、同步状态提示、消息中心
- 启动页与登录页品牌露出（Logo 展示）

## 品牌露出规则（当前阶段）
- Logo 仅放在启动页和登录页。
- Logo 资源路径：`/.trellis/assets/logo-polygon.png`
- 不允许在其他业务页面随意重复使用 Logo。
- 页面文案保持中文，且严禁 emoji。

# Impeccable Design Context

## Product and Audience
- 产品：LPG 配送员端 Web 原型（登录、工作台、快速开单、待配送完单、同步队列、安检、日结）。
- 主要用户：一线配送员与站点值班人员，常在移动设备（390x844 视口）与弱网环境下完成高频操作。
- 典型场景：站点与路途中单手操作，需快速完成关键动作并可回退。

## Brand Voice
- 稳定
- 可信
- 高效

## Intended Interface Feel
- 清晰、克制、可预期。
- 主操作强引导，错误反馈可定位。
- 移动端优先，减少视觉噪音与布局跳变。

## Visual References
- 当前项目 `ui-manifest`：浅色背景、卡片化布局、大圆角、弱边框、主色 `#4799a0`。
- 关键交易动作支持明显反馈（加载中/成功/失败/可重试）。

## Anti-References
- 不使用 emoji、重阴影、渐变大背景。
- 不做大范围视觉重写导致流程认知变化。
- 不打断既有业务路径（接口契约与流程保持不变）。

## Current Constraints
- 本轮仅允许最小安全改动，优先文档化审查结果。
- 现阶段保持原生 HTML/CSS/JS 结构，不做框架级重构。
